Toolu kullanamk için öncelikle go modülünü bilgisayarınıza indirmeniz gerekmektedir. https://go.dev/dl/ bu link üzerinden indirebilirsiniz.
Repoyu indirmek için;
git clone https://github.com/hacywhatt/dorkful.git

Daha sonra modülü başlatmak için;
cd dorkful

Toolu çalıştırmak için CMD ekranında;
go run main.go 
komutu çalıştırılmalıdır. Daha sonra http://localhost:8080 adresine gidilerek tool kullanılabilir.
